В текущей директории создать проект для сравнения различий схем 2х и более СУБД.
Язык: kotlin
Многомодульный Gradle проект. Каждый модуль должен являться git submodule.
Для начала создать 3 модуля: core, postgresql, jetbrains-plugin
Для модуля core создать первичную реализацию сравнения схем СУБД. Реализация не должна зависеть от конкретной СУБД.
Для модуля postgresql создать первичную реализацию сравнения схем специфичную для PostgreSQL. Через jdbc должны отправляться запросы к служебным таблицам postgresql
Для модуля jetbrains-plugin создать шаблон плагна для Jetbrains IDE с одним Application сервисом без реализации.
